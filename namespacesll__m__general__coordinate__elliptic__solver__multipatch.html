Initialization for elleptic solver.

To have the function phi such that div( A grad phi ) + B grad phi + C phi = rho where A is a matrix of functions , B a vectorial function, and C and rho a scalar function. A, B, C, rho can be discret or analytic. phi is given with a B-spline interpolator
