sll_m_quintic_splines Module Reference
sll_splines

Description

Provides capabilities for data and derivative interpolation with quintic splines.

sll_s_inspl5 and spln5 routines come from "An algorithm for the interpolation of functions using quintic splines" by E.H. Mund, P. Hallet and J.P. Hennart Journal of Computational and Applied Mathematics, volume 1, number 4, 1975. Periodic boundary conditions are not implemented. You must set function value and its derivative a the boundary
