sll_m_general_coordinate_elliptic_solver Module Reference
sll_general_coordinate_elliptic_solvers

Description

This module is an implementation of a general elliptic solver using Finite Element Methods with B-Splines used as the basis shape functions.

Just a reminder about bsplines

\( m+1 \) knots \( t_i \) in \( [0,1] \) with

\[ 0 \le t_0 \le t_1 \le \ldots \le t_m \le 1 \]

the n degree spline curve is

\[ \mathbf{S}(t) = \sum_{i = 0}^{m-n-1} b_{i,n} (t) . \mathbf{P}_{i} \,,\, t \in [0, 1], \]

where \( P_i \) is a polynomial function \( (m-n) \) points.

\( m-n \) B-splines n degree functions are defined recursively :

\[ b_{j, 0}(t) := \left\{ \begin{array}{ll} \mathrm{if} \quad t_j \leqslant t < t_{j+1} & 1 \\\ \mathrm{else} \quad & 0 \end{array} \right. \]

\[ b_{j,n}(t) := \frac{t-t_j}{t_{j+n}-t_j} b_{j,n-1}(t)+\frac{t_{j+n+1}-t}{t_{j+n+1}-t_{j+1}}b_{j+1,n-1}(t). \]

Elliptic solver on 2d curvilinear mesh

This solver works with analytical and discrete coordinate transformations.

Assemble the matrix for elliptic solver.

To have the function phi such that

\[ \nabla \cdot ( A \nabla \phi ) + B \nabla \phi + C \phi = \rho \]

where

  • A is a matrix of functions ,
  • B a vectorial function,
  • C a scalar function.
  • rho a scalar function.
  • A, B, C, rho can be discret or analytic.
