sll_particle_groups

Description

Create particle groups.

Author
Katharina Kormann, IPP, and Martin Campos Pinto, CNRS

this library contains several particle groups in 1D1V and 2D2V. The lbf particle group implements deterministic (grid sampled) particles with a resampling capability using the LBF (local backward flow) reconstruction technique.
