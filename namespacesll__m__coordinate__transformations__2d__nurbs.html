sll_m_coordinate_transformations_2d_nurbs Module Reference
sll_coordinate_transformations

Description

Functions for the general transformation defined with nurbs*.

The nurbs-based coordinate transformation is associated with 2 logical meshes: the 'minimal' logical mesh, which is implicit in the spline description and that is able to faitfully represent a given geometry. The other logical mesh, which is the usual logical mesh on which we want to represent quantities. This needs to be present to answer questions like Tx_node(i,j). We set this logical mesh outside of the read_from_file routine.
