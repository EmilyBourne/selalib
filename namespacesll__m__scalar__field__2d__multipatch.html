sll_m_scalar_field_2d_multipatch Module Reference

Description

Encapsulates a group of scalar fields, each associated with a particular region of a physical domain. The ensemble is referred to as 'multipatch'.

Author
  • Edwin

The use of the 2D scalar field, multipatch version, is a direct extension of the use of the 2D scalar field, except that its interface routines include and additional parameter, the patch number under inquiry (a number between 0 and the number of patches - 1) plus additional information related with the connectivity between the patches.
